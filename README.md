# Ecommerce-Customer-Behavior-Analysis
E-commerce customer behavior analysis project using Python. Explores spending patterns, membership impact, and customer satisfaction through data visualization and hypothesis testing. Provides business insights and recommendations to improve customer engagement and revenue.
# E-commerce Customer Behavior Analysis

## 📌 Objective
To analyze customer behavior and identify key factors influencing spending and satisfaction.

## 📊 Dataset
The dataset contains customer demographics, spending patterns, and satisfaction levels.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Statistical Testing (T-test)

## 🔍 Key Insights
- Gold members contribute the highest revenue
- Discounts influence customer spending
- Satisfied customers tend to spend more
- Identified inactive customers for retention
